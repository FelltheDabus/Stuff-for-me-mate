# To get the whole repository:

> Open the explorer and create a folder for you to store the things. It will need to be an empty folder for the clone to work

> Right click inside the folder --> open in terminal

> Click the green code button in the top right of the GitHub page and copy the HTTPS adress

> Go back to the terminal and enter "git clone (paste the HTTPS adress)"

> Watch the magic happen


# To get specific files:

> You can just click on individual files you want here on GitHub and then click the download up on the top right next to where it says "RAW"
