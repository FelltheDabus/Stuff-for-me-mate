# Hey mate, read this!

## To get specific files:

> You can just click on individual files you want here on GitHub and then click the download up on the top right next to where it says "RAW"

## To get the whole repository:

> Click the green code button in the top right of the GitHub page and copy the URL

> Open the explorer and create a folder for you to store the things. It will need to be an empty folder for the clone to work

> Right click inside the folder --> open in terminal

> Enter `git clone (paste the URL)`

> Watch the magic happen

# Also, if you want to be a smooth cool person

## Windows

### To see all packages Winget knows about -  a more comprehensive list than you would get from the settings GUI app

> `winget list`

### To update everything on your computer.

> `winget upgrade --all`

## Linux

### To update everything on your compute

> `sudo apt update && sudo apt upgrade`
